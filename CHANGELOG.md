# R6SSS Discord Bot Changelog


## v2.5.6 (2025/12/30)

### 🔁 変更
- サーバーステータス埋め込み更新処理を別の関数へ分離
  - 設定関連コマンドの実行時にサーバーステータス埋め込みの更新を行うよう改良
- 一部テキストを改善

### 🔧 修正
- create コマンドでのサーバーステータスメッセージ新規作成時に、メンテナンススケジュールの埋め込みが同時に送信されない問題を修正


## v2.5.5 (2025/12/29)

### 🟢 追加
- サーバーステータス定期更新時にデータに変化があるかチェックする処理を実装
  - サーバーステータスとメンテナンススケジュールの各値が前回チェック時と異なっているかをチェックして、異なっている場合のみ更新を実行する


## v2.5.4 (2025/12/29)

### 🔁 変更
- メンテナンススケジュールの警告テキストを変更
- コマンド実行時エラーのメッセージと処理を改良
- サーバーステータス定期更新処理の例外処理を改良
- チャンネル取得等のメソッドを `get_or_fetch` メソッドへ置換
- ログイン完了時に Uptime Kuma へ Ping を送信するように
- on_connect イベントを削除
- 依存関係の更新
- 不要なコードを削除

### 🔧 修正
- サーバーステータス変更通知メッセージのプラットフォーム名表記を修正
- ロガーの挙動を修正


## v2.5.3 (2025/09/14)

- 支援メッセージの追加
- 起動時のログファイル生成処理の修正


## v2.5.2 (2025/08/27)

- サーバーステータス更新処理の例外処理を実装
- Uptime Kuma の Ping 送信時の例外処理を修正


## v2.5.0 (2025/06/20)

- ステータスに応じて埋め込みメッセージの色を変える機能を実装
- メンテナンススケジュールの多言語対応
- サーバーステータス通知メッセージの「影響を受けていた機能一覧」を設定言語に応じて翻訳されるよう修正


## v2.4.0 (2025/06/15)

- 通知メッセージが誤った表示言語で送信される問題を修正
- /create コマンドのエラーハンドリングを改善
- その他デバッグ用の改善


## v2.3.2 (2025/06/11)

- スケジュールの埋め込みメッセージが正常に生成されない問題を修正


## v2.3.1 (2025/06/11)

- サーバーステータス通知メッセージの「サーバーステータスを表示」の表記を削除


## v2.3.0 (2025/06/10)

- サーバーステータスに変化があった際に送信される通知メッセージに計画メンテナンス用の通知を追加
- メンテナンススケジュールのダウンタイム表記を「約 n 分」に変更
- コードの整理


## v2.2.0 (2025/06/08)

- サーバーステータス通知メッセージに挿入されるリンクの修正
- /create コマンドで内部エラーが発生して使用できない問題を修正
- 内部エラー発生時のエラーメッセージにエラーコードを追加
- データベース接続関連の修正


## v2.1.1 (2025/06/08)

- /schedule の説明テキストを追加
- 使用していない管理用コマンドを削除
- ロガーの改良


## v2.1.0 (2025/06/05)

- サーバーステータスメッセージのメンテナンススケジュール表示設定 (/setscheduledisplay) を実装
- 各コマンドの実行時にサーバーステータスの更新を行わないよう修正
- サーバーステータス埋め込みメッセージ定期更新処理の最適化
- コードのリファクタリング


## v2.0.0 (2025/05/31)

- サーバーステータスの埋め込みメッセージのデザインを刷新
- メンテナンススケジュールの表示機能を実装
- サーバーステータスが変更された際に通知メッセージを送信する機能を実装

- 各サーバーの設定情報をJSON形式のファイルから MongoDB へ移行
- 多言語対応の改善
- ライブラリーの更新
- コードのリファクタリング
